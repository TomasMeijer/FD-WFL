# From a Frequency-Domain Willems' Lemma to Data-Driven Predictive Control

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Overview

This repository provides the **MATLAB code** for all examples in the paper ``From a Frequency-Domain Willems' Lemma to Data-Driven Predictive Control''
It includes the following examples:

- Example 1: FRF-based simulation using noise-free data
- Example 2: FRF-based simulation using noisy data
- Example 3: FRF-based LQR
- Numerical case study: Frequency-domain data-driven predictive control (FreePC)
